# LROP - Игровое меню с мини-играми

Консольное приложение на языке C с набором игр (Счастливчик, Найди пары, Сапёр, Крестики-нолики, Тетрис).

**Полная документация**: См. [README.md](README.md)

## Быстрый старт

```bash
make
./game_menu
```

## Игры

1. **Счастливчик** - Угадай число
2. **Найди пары** - Игра "Мемори"
3. **Сапёр** - Классический минный тральщик
4. **Крестики-нолики** - Против компьютера
5. **Тетрис** - Собери линии

Все игры поддерживают 3 уровня сложности.
